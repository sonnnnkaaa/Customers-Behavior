# Кластеризация данных о поведении клиентов онлайн-магазинов

Цель: выявить основные группы клиентов, исходя из их поведения методом K-Means

## Описание

В этом репозитории содержатся код, данные и визуализации, связанные с проектом. У Вас есть возможность протестировать код на своей машине или просто просмотреть его вместе с результатами. 

**Шаги:**
*   Провести разведочный анализ данных (EDA).
*   Построить модель K-Means для выделения групп объектов.
*   Интерпретировать полученные результаты.

## Используемые библиотеки

Проект написан на Python с использованием следующих основных библиотек:
*   `pandas`, `numpy` — для обработки данных
*   `matplotlib`, `seaborn` — для визуализации
*   `scikit-learn`, `scipy` — для преобразования данных, построения модели K-Means и проведения статистического теста

Полный список зависимостей указан в файле `requirements.txt`.
